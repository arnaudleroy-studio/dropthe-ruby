dropthe-ruby

Ruby gem for DropThe open data API. Query movies, series, crypto, companies, and people from 1.83M entities.

About DropThe

This library provides idiomatic Ruby bindings for the DropThe open data API. DropThe is a data utility media network that aggregates and structures information across entertainment, finance, and technology.

The DropThe platform maintains a knowledge graph of 1.83 million entities with 2.9 million typed relationships, covering movies, TV series, cryptocurrencies, public companies, and notable people. This Ruby library provides full access to search, entity retrieval, and graph traversal.

# Installation

Ruby 3.0+ | Rails 6+ 
# Install the gem
gem install dropthe

# Or add to Gemfile
gem 'dropthe'

# Quick Start

require 'dropthe'

# Initialize the client
client = Dropthe::Client.new(api_key: 'your-api-key')

# Search 1.83M entities
results = client.search('Breaking Bad')
results.each do |entity|
  puts "#{entity.name} (#{entity.type})"
end

# Get entity with knowledge graph links
movie = client.get_entity('the-dark-knight-2008')
puts movie.name
puts movie.data['director']

# Browse by type with filters
crypto = client.list(type: 'cryptocurrencies', limit: 10)
crypto.each { |c| puts "#{c.name}: #{c.data['price_usd']}" }

# Features

  • Thread-safe, Rails-compatible, lazy enumeration
  • Movies & series metadata
  • Cryptocurrency tracking
  • Company profiles
  • People database

# API Reference

GET client.search(query, **options) -> Array<Entity>

Full-text search across movies, series, crypto, companies, and people datasets.

GET client.get_entity(slug) -> Entity

Retrieve a single entity with all metadata, data fields, and linked entities.

GET client.list(type:, limit:, offset:) -> Array<Entity>

Paginated listing of entities filtered by type with sorting options.

GET client.get_links(entity_id) -> Array<EntityLink>

Access the knowledge graph: 2.9M+ typed relationships between entities.
