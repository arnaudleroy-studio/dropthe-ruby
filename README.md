# DropThe

Ruby client for [DropThe](https://dropthe.org) -- open data API for movies, series, crypto, companies, and people datasets.

[![Gem Version](https://badge.fury.io/rb/dropthe.svg)](https://rubygems.org/gems/dropthe)

## Install

```
gem install dropthe
```

Or add to your Gemfile:

```ruby
gem 'dropthe'
```

## Usage

```ruby
require 'dropthe'

puts DropThe::VERSION
puts DropThe::BASE_URL
```
